1. 인라인 코드
   백틱을 사용하여 작성<br>
   ex)
   &#96;인라인코드 작성&#96;
   <br>
   결과  
    `인라인코드 작성`

2. 코드 블럭
   백틱 3개를 사용하여 작성<br>
   ex)<br>
   &#96;&#96;&#96;<br>
   코드블럭 작성<br>
   &#96;&#96;&#96;
   <br>  
   결과
   ```
   코드블럭 작성
   ```
3. 코드 하이라이트
   백틱 3개 뒤에 언어 작성<br>
   ex)<br>
   &#96;&#96;&#96;javascript<br>
   자바스크립트 코드블럭 작성<br>
   &#96;&#96;&#96;
   <br>
   결과
   &#96;&#96;&#96;
   ```javascript
   자바스크립트 코드블럭 작성
   ```
