nestjs 프로젝트 셋업

1. 프로젝트 생성

   ```javascript
   nest g application 이름
   ```

2. package.json 작성된 패키지 불러오기

   ```javascript
   npm i
   ```

3. 실행

   ```javascript
   npm run start:dev
   ```
