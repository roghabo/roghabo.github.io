### React Helmet

---

- title을 바꿔주는 역할을 함

- 설치
  ```
  npm i react-helmet
  npm i @types/react-helmet
  ```

<br><br>

- 사용 예시

  ```html
  import Helmet from "react-helmet"

  <Helmet>
    <title>Roghabo</title>
  </Helmet>
  ```
